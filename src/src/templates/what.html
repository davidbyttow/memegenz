Memegen is an internal, Square-only accessible application that allows anyone to create a meme.


Steps to success:

  • 1. Select or Upload a Meme Template
  • 2. Type something topical and funny
  • 3. Share it with your peeps
  • 4. Profit


    There are many more features in the works:

    • - Better +1 buttons (ability to unvote)
    • - Animated GIF support
    • - Vote email notifications


    Doesn't this exist at Google?
    Yes. This is built from the ground-up for Square. And we can can take it wherever like.


    Can I contribute?
    Yes! The code is all here: https://git.squareup.com/d/memegen



    Email d@squareup.com to send feedback or suggestions!
